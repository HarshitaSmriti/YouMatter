# YouMatter

AI-powered mental wellness companion built using React, Vite, Supabase, and Express.

## Features

- AI Chat Support
- Mood Tracking
- Journal System
- Breathing Exercises
- Crisis Support
- Google Authentication
- Dark Mode

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Express.js
- MongoDB

## Run Locally

```bash
npm install
npm run dev
